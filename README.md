
# Wheelchair accesibility of the Budapest Public Transport Network

Objective: To gather and analyze data on BKK tram, bus and metro routes, focusing on Wheelchair accessibility. Specifically, using data from the BKK API, determine what share of the BKK network is accessible to people in wheelchairs. Given a list of wheelchair accessible stations, generate data on which stations can be reached from those starting points.

Key Concepts and tools: Data from APIs, requests, data analysis.
